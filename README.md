Welkom bij de Simac-IT-NL Github omgeving. Lees deze README.md door voor de [code-of-conduct](https://en.wikipedia.org/wiki/Code_of_conduct)

## Policies
Als je niet voldoet aan de onderstaande policies kan je account verwijderd worden van de organisatie. Hoe clean-ups zullen gebeuren word later bepaald. Naar alle waarschijnlijkheid zal in de toekomst gemigreerd worden naar Azure Entra-ID accounts

### User policies
Je mag een account hebben binnen de github organisatie:
- Je naam en achternaam of werkmail bekend zijn. **Als je deze informatie niet publiekelijk wilt delen via je account zie "Anoniem blijven?"**
